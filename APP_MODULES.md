# 训练助手 App 模块说明

力量训练 PWA，移动端优先（480px），浅色/深色双模式，React + Supabase。

## 底部 5 个 Tab

| Tab | 功能 |
|-----|------|
| **今日** | 核心页面。根据激活计划自动生成今日训练内容，区分休息日/训练日/已完成三种状态。一键开始全屏训练会话，支持身体数据快速打卡。 |
| **训练** | 计划库。浏览内置计划、启用/暂停/结束计划、创建自定义计划、动作库。 |
| **饮食** | 营养管理。配置热量缺口与宏量素目标，记录每日餐食与有氧运动，进度条追踪。 |
| **数据** | 三大子页：力量趋势折线图、身体指标图表、训练日历（月视图，可追溯/删除记录）。 |
| **我的** | 账号管理（匿名/邮箱）、主题切换、器械配重配置（杠铃片/哑铃/龙门架）、训练画像。 |

## 核心训练流

**画像引导（5步）** → **启用计划** → **今日页面**展示当日动作（T1红/T2蓝/T3紫，含重量×次数×组数） → **全屏训练会话**（逐组打卡、RPE、节奏、组间计时、杠铃片可视化） → **保存**写入日历。

## 设计关键

- 主色 `#FF6B35`，T1/T2/T3 为红/蓝/紫
- 按钮 44px/12px 圆角，卡片 16px 圆角
- 线性图标（lucide-react），强字重标题
